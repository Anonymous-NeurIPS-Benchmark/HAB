# License and Access Restrictions

Unless otherwise noted, HAB annotations, metadata, prompts, dataset splits, hardware dictionary entries, and evaluation labels are intended to be released under the Creative Commons Attribution 4.0 International license (CC BY 4.0).

The HAB-REF evaluation code and scripts are intended to be released under the MIT License.

Third-party Arduino project text, source code, and reference sketches remain under their original licenses. These materials should be redistributed only when the original license permits redistribution. For projects with unclear redistribution rights, HAB should include source URLs, annotations, and evaluation metadata but should not redistribute the original project text or code.

A final release should include a per-instance provenance and license audit file specifying the source URL, license status, and redistribution status for each project.
